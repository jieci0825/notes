# Linux

## 基础命令
- **pwd**：显示当前路径
- **ping**:  测试地址
- **rm**：删除文件和目录
  - -r：递归删除，删除目录时使用
  - -f：强制删除，忽略任何警告
- **cat**：查看文件
- **ls**：列出当前目录可见文件
  - -l: 显示详细信息
  - -a: 显示所有文件，包括以 . 开头的隐藏文件
- **cp**：复制文件或目录。语法：cp [选项] 源文件... 目标路径
  - 目标路径可以是两个情况：
    - 一个目录（将源文件复制到此目录下）。
    - 一个新的文件名（复制的同时进行重命名）。
  - 复制整个目录（需要 -r或 -R选项）。将 my_folder整个目录复制到当前位置，并命名为 my_folder_backup。命令为：`cp -r my_folder my_folder_backup`
- **mv**：移动或重命名
  - mv a aaa: 将 a 重命名为 aaa
- **mkdir**：创建目录
  - **-p**：添加 -p 穿透式创建。/a/b/c 。如果不存在，则会创建多层级。
  - **{a,b}**: 同时创建 a 和 b 两个目录。
- **touch**: 创建空白文件
- **head**: 默认查看文件前 10 行。可以指定数字实现指定行数：`head -1 abc.js`
- **tail**: 和 head 相反，是查看末尾的。
- **打包**: `tar -jcvpf [打包后的文件].bz2 [要打包的文件路径]`。打包的时候，可以在 -jcvpf 后面跟上参数 `--exclude=xxx` 可以排除这些目录，也可以是文件 。
- **解包**: `tar -jxvpf [要解包的文件] [解包到哪里]`。解包到哪里省略的话，就是当前目录。
- **cd ～**：切换到当前用户的家目录
- **cd /**: 切换到文件系统的根目录

## Linux-ssh

- **登录**： `ssh user@111.229.150.139`
- **退出登录**: `exit` 
- **传输文件**： `scp bg1.png root@111.229.150.139:/test-temp`。冒号 : 后面的部分表示远程服务器上的目标路径。这里是把文件复制到服务器的根目录下的 /test-temp 文件夹中。如果把文件路径放到目标机子的后面，则会变为**下载行为**。`scp root@111.229.150.139:/test-temp/bg1.png ./` 把这个目标机的文件拉取到当前目录。
- **生成密钥**: `ssh-keygen`。这条命令是在客户端生成的。即你本地设备为 b，链接的设备为 a。那么在 b 上执行。
- **将密钥文件传输**:  `ssh-copy-id -i /Users/coderjc/.ssh/id_ed25519.pub root@111.229.150.139`。生成的密钥文件成功之后，将后缀 .pub 的密钥文件传递给设备 a。传递这个密钥文件之后，后续在登录，就不要再输入密码了。、
- **rsync**: 高效的文件同步与复制。
  - **本地 -> 远程**: `rsync -av /src/ user@host:/dest/`
  - **本地 -> 本地**: `rsync -av /src/ /dest/`
  - **远程 -> 本地**: `rsync -av user@host:/src/ /dest/`
  - 参数说明：
    - **-a**: 一般同步目录常用
    - **-v**: 输出详细信息，用于调试和观察同步行为。
    - **-z**: 传输中压缩数据，适合网络环境较差或传输数据量大的情况。

## 软件和包的管理

### rpm
-  **rpm -qa**: 列出系统中所有已安装的 RPM 软件包。 `rpm -qa | grep curl`，从所有已安装的软件包中筛选出名字包含 curl 的那些
  - -q：query（查询）
  - -a：all（全部）
- **rpm -ql <package-name>**: 查询系统中某个已安装的 RPM 包，并输出该包安装到系统中的全部文件列表。
- **rpm -qi <package-name>**: 输出指定已安装软件包的完整元信息，包括版本、发行信息、安装时间、打包者、摘要描述等。
- **rpm -qf <file-path>**: 告诉你指定的文件是由哪个 RPM 包安装到系统中的。
- **rpm -e <package-name>**: 从系统中卸载一个已安装的 RPM 包。
- **rpm -ivh xxx.rpm**:  安装本地 .rpm 文件。不会自动处理依赖
  - -i = install
  - -v = verbose（显示过程）
  - -h = hash（显示进度条）
