# vim 初体验

## 模式
- **正常模式**: 刚打开 Vim 时的默认模式。用于移动光标、复制粘贴、删除文本等。不能输入文字。
- **插入模式**: 按 `i` 键进入。在此模式下可以像普通编辑器一样输入文字。无法执行退出命令。
- **可视模式**: 按 `v`, `V`, `Ctrl+v` 进入。用于选择文本块。
- **命令行模式**: 在正常模式下按 `:`进入。屏幕底部会出现一个冒号 :，等待你输入命令（如保存、退出）。

## 常用命令
- **:q**: 退出
- **:w**: 保存
- **:wq**: 保存并退出
- **w!**: :强制保存
- **q!**: :强制退出
- **wq!**: :强制保存并退出
- **u**: 撤销
- **gg**: 移动到文件首行
- **G**: 移动到文件末尾行。或者输入 `:$` 回车也是文件末尾行。
- **0**: 移动到当前行的行尾
- **$**: 移动到当前行的行尾。如果不行则使用 `shift + 4`
- **:100**: 移动到第一百行
- **/{任意字符]**: 比如 `/name` 就是查找 name 字符。如果 name 存在多个。按下 n 就可以往下翻动。大写的 N 就是向上。
- **yy**：复制当前行
- **y4y**: 复制从当前行开始算的后面四行。
- **p**: 粘贴
- **dd**：删除当前行
- **d2d**: 删除从当前行开始算的后面 2 行
- **d0**: 删除光标的位置开始到行首位置的字符。
- **d$**: 删除光标的位置开始到行尾的位置的字符。如果不生效可以试试 `shift + d`
- **dG**: 删除从当前行到文件末尾行的内容。反之是 `dgg`。
- **v**: 进入可视模式，ctrl + v 进入可视块模式。
  - 选择对应的内容 + d：是删除选中的
  - 选择对应的内容 + Y: 是复制选中的
- **x**: 退格删除一个字符
- **u**: 回退上一步的操作
- **:1,$  s/abc/ABC/g**: 从第一行到最后一行。将 abc 替换为 ABC。添加 g 防止一行出现多个 abc 时，只替换了这行的第一个 abc。
